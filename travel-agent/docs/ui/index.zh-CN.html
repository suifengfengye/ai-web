Travel Agent UI / 中文版

旅游助手产品 UI 中文设计稿

这一版是基于你现有 `02-ui-design.md` 方案的中文版静态设计稿，保留原有视觉风格，但把首页、桌面端规划页、移动端规划页里的核心内容改成中文，方便你直接拿去做产品评审、业务讨论和前端开发对接。

视觉方向

延续青蓝和绿色为主的清爽风格，突出地图感、规划感和可信度，适合 AI 旅游助手的产品定位。

布局策略

桌面端采用左侧对话、右侧结构化结果的双栏布局；移动端保留摘要、快捷改写、单日行程和天气预算卡片。

使用场景

适用于产品需求确认、UI 讨论、前端还原参考，也适合后续继续转成 Next.js 页面。
