Minimal + Fresh Travel Style

Upgrade static design boards into a real Travel Agent HTML prototype

This version replaces image-only mockups with an actual HTML prototype. The information architecture is split into a web chat experience for travelers, a web admin console for RAG knowledge management, and a mobile chat flow for on-the-go itinerary changes.

Open Web Prototype Open Admin Console
Design Mood

Minimal, calm, trustworthy, and airy with a travel-friendly tone.

Output Type

Native HTML and CSS only, ready to evolve into React, Vue, or Next.js.

Core Scenarios

Trip planning chat, knowledge base operations, and mobile itinerary follow-up.

Visual Direction

Soft mint, sky, and white surfaces create a fresh travel feeling, while spacing and card hierarchy remove the clutter and overlaps from the previous SVG layout.

Interaction Focus

The traveler view prioritizes chat, recommendations, weather, and budget. The admin view prioritizes upload, update, indexing, and retrieval quality management.

Responsive Strategy

Desktop uses structured two-column layouts. Mobile keeps a compact single-column chat flow with quick rewrites and daily itinerary cards.
