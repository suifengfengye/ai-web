# Agent Video 项目规则

## 适用范围

- 适用于当前仓库内所有以“数字th-”开头的期数目录。
- 默认每个期数目录的入口文件为 `agent.py`。

## 技术与环境规则

- Python 版本按 3.12 约定执行。
- 技术栈按 langchain 1.2.0 与 langgraph 1.05 约定执行。
- 运行 Python 命令时优先使用 conda 环境 `agent-video`。

## 强制验证规则

- 只要本次任务修改了任意 `*th-*` 目录内代码，回复前必须完成运行验证。
- 验证命令为在对应期数目录执行 `python agent.py`。
- 若无法直接激活 conda，使用 `conda run -n agent-video python agent.py` 执行验证。
- 若修改涉及多个期数目录，必须逐个目录执行验证，不得只验证其中一部分。
- 任一目录验证失败时，不得宣告任务完成，需先修复并重新验证。

## 结果回报规则

- 最终反馈必须包含本次已验证的目录列表。
- 最终反馈必须包含每个目录的执行命令。
- 最终反馈必须明确每个目录的结果状态（成功或失败）及关键报错信息。
